# README

## 需求和功能

### 1. 基础需求

- 生成一个棋盘
- 根据难度生成一个可解的数独游戏
- 提供用户输入接口，允许用户填写数独
- 检验用户输入是否正确

### 2. 扩展功能

- 多语言支持
- 提示功能
- 保存数据读取

## 程序的框架

### 1. 程序目录

- `src/` - 源文件
- `include/` - 头文件
- `tests/` - 测试代码
- `bulid/` - 构建代码

### 2. 主要模块

- `Main` - 主函数
- `Scene` - 棋盘的生成，加载和数独逻辑
- `Input` - 用户输入接口
